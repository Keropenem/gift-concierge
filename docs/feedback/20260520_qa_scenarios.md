# トラブル時の確認ポイント

| 症状 | 確認場所 |
|---|---|
| Feedback保存できない | Supabase `feedback` テーブル / RLS Policy |
| プロンプト履歴が残らない | Supabase `prompt_history` テーブル / RLS Policy |
| URLチェックで全URLが「到達不可」になる | Vercel の fetch outbound、User-Agent ヘッダー |
| 受け手データがLLM応答に反映されない | route.ts の初回メッセージ contextParts のログ |
| 「光齋様お久しぶり」が出続ける | プロンプト本体（/admin/prompt）の改修要 |
| 履歴の最初に黒バブルでシステム文が見える | `app/api/chat/session/route.ts` のフォールバック確認 |
